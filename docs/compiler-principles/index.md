# 编译原理

## 课程概述

编译原理是计算机科学的核心课程之一，主要研究如何将高级程序语言翻译成机器语言的理论、技术和工具。

## 课程目标

- 理解编译过程的各个阶段及其相互关系
- 掌握词法分析、语法分析等核心技术
- 学习语法制导翻译和中间代码生成
- 了解运行时环境的组织和管理
